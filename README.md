
## [Operator](https://github.com/guyanyijiu/operator)


> 对接第三方平台获取用户运营商处的数据
- 闪银 Wecash
- 魔蝎 Moxie


### 通过composer安装

编辑 **composer.json** 添加如下内容
```
"repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/guyanyijiu/operator"
        }
    ]
```
然后执行如下命令
```
$ composer require guyanyijiu/operator
```
